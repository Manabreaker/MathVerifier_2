Примеры математических доказательств

∫

Теорема о среднем значении

Если функция $f(x)$ непрерывна на отрезке $[a,b]$ и дифференцируема на интервале $(a,b)$, то существует точка $c \in (a,b)$ такая, что:

$$f(b) - f(a) = f'(c)(b-a)$$

∑

Сумма арифметической прогрессии

Для арифметической прогрессии с первым членом $a$ и разностью $d$ сумма $n$ первых членов равна:

$$S_n = \frac{n}{2}(2a + (n-1)d) = \frac{n}{2}(a + a_n)$$

π

Метод математической индукции

Докажем, что $1 + 2 + 3 + \ldots + n = \frac{n(n+1)}{2}$ для всех $n \in \mathbb{N}$.

Для $n=1$: $1 = \frac{1(1+1)}{2} = 1$ ✓

Предположим, формула верна для $n=k$: $1 + 2 + \ldots + k = \frac{k(k+1)}{2}$

Для $n=k+1$: $1 + 2 + \ldots + k + (k+1) = \frac{k(k+1)}{2} + $

$(k+1) = \frac{(k+1)(k+2)}{2}$ ✓
